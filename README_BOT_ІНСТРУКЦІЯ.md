# ІНСТРУКЦІЯ З ЗАПУСКУ (БОТ + WEB APP)

Наш проєкт тепер складається з **ТРЬОХ** окремих серверів, які мають працювати **ОДНОЧАСНО** у **ТРЬОХ** окремих терміналах.

1.  **СЕРВЕР 1: Web App (Flask)** - Наш "мозок", сайт з аналітикою.
2.  **СЕРВЕР 2: Тунель (ngrok)** - "Міст" з Інтернету до нашого СЕРВЕРА 1.
3.  **СЕРВЕР 3: Бот (aiogram)** - Наша "вхідні двері" у Telegram.

[Зображення діаграми архітектури з трьома компонентами: Термінал 1 (Flask), Термінал 2 (ngrok) та Термінал 3 (Telegram Bot), що показує, як вони з'єднані]

---

## КРОК 1. Налаштування (Робимо 1 раз)

1.  **Отримайте Токен Бота:**
    * Відкрийте Telegram, знайдіть `@BotFather`.
    * Надішліть йому команду `/newbot`.
    * Дотримуйтесь інструкцій, дайте йому ім'я.
    * `@BotFather` надішле вам токен (довгий рядок).

2.  **Оновіть `.env`:**
    * Скопіюйте цей токен і вставте його у ваш `project/.env` у **Секцію 8** (`BOT_TOKEN=...`).
    * **ВАЖЛИВО:** Переконайтеся, що `BASE_URL` у `.env` – це ваша **актуальна `ngrok` адреса**.

3.  **Встановіть нову бібліотеку:**
    * Відкрийте термінал у папці `project/`.
    * Активуйте `venv`: `.\venv\Scripts\activate`
    * Виконайте: `pip install -r requirements.txt` (щоб встановити `aiogram`).
